# Shotcount Wallpaper Hangers

A concierge chat application for Shotcount Wallpaper Hangers — a professional wallpaper installation and removal service in Washington, DC.

## Setup

**Prerequisites:** Node.js

1. Install dependencies:
   ```
   npm install
   ```
2. Copy `.env.example` to `.env` and set your `OPENAI_API_KEY`
3. Build and preview:
   ```
   npm run build
   npm run preview
   ```
